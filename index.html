STEP 01 — 스프레드시트 ID 확인
Google Sheets 편집 화면 URL에서 ID 복사
웹에 게시된 URL(2PACX-...)이 아닌, 평소 편집할 때 보이는 일반 편집 URL이 필요합니다.
편집 URL 예시: https://docs.google.com/spreadsheets/d/1BxiMVs0XRA5nFMdKvBdBZjgmUUqptlbs74OgVE2upms/edit ↑ /d/ 와 /edit 사이의 노란 부분이 Sheet ID입니다
STEP 02 — 탭별 수식 입력 (각 탭 A1 셀에)
[WTI 탭] =GOOGLEFINANCE("NYSEARCA:USO","close",TODAY()-60,TODAY(),"DAILY") [Brent 탭] =GOOGLEFINANCE("NYSEARCA:BNO","close",TODAY()-60,TODAY(),"DAILY") [USDKRW 탭] =GOOGLEFINANCE("CURRENCY:USDKRW","close",TODAY()-40,TODAY(),"DAILY") [CNYKRW 탭] =GOOGLEFINANCE("CURRENCY:CNYKRW","close",TODAY()-40,TODAY(),"DAILY") [EURKRW 탭] =GOOGLEFINANCE("CURRENCY:EURKRW","close",TODAY()-40,TODAY(),"DAILY")
STEP 03 — 시트 공유 설정 (웹에 게시 불필요!)
링크 공유를 "링크가 있는 사용자 모두"로 설정
오른쪽 상단 공유 버튼 → "링크가 있는 사용자" → 뷰어 선택 후 저장
✓ "웹에 게시"는 필요 없습니다. 이 방법이 더 간단합니다.
편집 URL 전체를 붙여넣어도 자동 추출됩니다.
