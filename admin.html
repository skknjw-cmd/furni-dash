📋 Apps Script 설치 방법 보기

4. 배포 → 새 배포 → 웹 앱
   · 실행 계정: 나 (Me)
   · 액세스: 모든 사용자 (Anyone)
5. 배포 URL 복사 → 위 입력란에 붙여넣기 → 저장
